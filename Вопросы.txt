1. Какие http запросы существуют?
GET, PUT, PATCH, POST, DELETE
2. Какие запросы может отдавать html файл?
GET(запрашивает данные с определенного ресурса) и POST(отправляет данные на сервер для последующей их обработки)
3. Для чего нужен каждый запрос?
GET-получение информации с сервера, PUT-перезапись существующей информации на сервере,
 PATCH-частично изменяет ресурс, POST-создание новой записи на сервере, DELETE-удаление существующей информации на сервере
